# Mobile Keyboard Installation Guide

This guide explains how to install and use the Talkute keyboard on iOS and Android devices.

## iOS Installation

### Step 1: Install the App
1. Download Talkute from the App Store
2. Open the app and complete the initial setup
3. Grant microphone permission when prompted

### Step 2: Enable the Keyboard
1. Open **Settings** on your device
2. Go to **General** → **Keyboard** → **Keyboards**
3. Tap **Add New Keyboard**
4. Find and select **Talkute** from the list

### Step 3: Enable Full Access (Recommended)
For clipboard integration and voice processing:
1. Go to **Settings** → **General** → **Keyboard** → **Keyboards**
2. Tap on **Talkute**
3. Enable **Allow Full Access**

### Using the Keyboard
1. Open any app with a text field
2. Tap and hold the globe icon on your keyboard
3. Select **Talkute** from the list
4. Tap the microphone button to start voice input

---

## Android Installation

### Step 1: Install the App
1. Download Talkute from the Play Store
2. Open the app and complete the initial setup
3. Grant microphone permission when prompted

### Step 2: Enable the Keyboard
1. Open **Settings** on your device
2. Go to **System** → **Languages & input**
3. Tap **Virtual keyboard**
4. Find and enable **Talkute Keyboard**

### Step 3: Set as Default (Optional)
1. In **Virtual keyboard** settings
2. Set **Talkute** as the default keyboard

### Using the Keyboard
1. Open any app with a text field
2. Tap the text field to show the keyboard
3. Tap the microphone button to start voice input

---

## Features

- **Push-to-talk**: Hold the microphone button while speaking
- **Auto-detect language**: The keyboard detects your language automatically
- **Polishing levels**: Light, Standard, or Deep text polishing
- **Translation**: Automatic translation to your target language

## Troubleshooting

### Keyboard not appearing
- Restart the device and try again
- Check that the keyboard is enabled in system settings

### Voice input not working
- Ensure microphone permission is granted
- Try restarting the app

### Text not inserting
- iOS: Ensure Full Access is enabled
- Android: Check that Talkute is set as an input method

---

## Privacy

The Talkute keyboard:
- Only sends audio when the microphone button is pressed
- Does not log keystrokes
- Processes voice on secure servers
