# Mobile Keyboard Installation Guide

This guide explains how to install and use the Talkute keyboard on iOS and Android devices.

## iOS Installation

### Step 1: Install the App
1. Download Talkute from the App Store
2. Open the app and complete the initial setup
3. Grant microphone permission when prompted

### Step 2: Enable the Keyboard
1. Open **Settings** on your device
2. Go to **General** → **Keyboard** → **Keyboards**
3. Tap **Add New Keyboard**
4. Find and select **Talkute** from the list
5. The keyboard will now be available in your keyboard picker

### Step 3: Enable Full Access (Recommended)
For clipboard integration and voice processing:
1. Go to **Settings** → **General** → **Keyboard** → **Keyboards**
2. Tap on **Talkute**
3. Enable **Allow Full Access**
4. Confirm the permission dialog

### Using the Keyboard
1. Open any app with a text field
2. Tap and hold the globe icon on your current keyboard
3. Select **Talkute** from the list
4. Tap the microphone button to start voice input
5. Speak your text and release to insert

---

## Android Installation

### Step 1: Install the App
1. Download Talkute from the Play Store
2. Open the app and complete the initial setup
3. Grant microphone permission when prompted

### Step 2: Enable the Keyboard
1. Open **Settings** on your device
2. Go to **System** → **Languages & input**
3. Tap **Virtual keyboard** or **On-screen keyboard**
4. Find and enable **Talkute Keyboard**
5. A warning will appear - tap **OK** to confirm

### Step 3: Set as Default (Optional)
1. In **Virtual keyboard** settings
2. Tap **Manage keyboards**
3. Set **Talkute** as the default keyboard
4. Or use the keyboard picker when needed

### Using the Keyboard
1. Open any app with a text field
2. Tap the text field to show the keyboard
3. Tap the microphone button to start voice input
4. Speak your text and release to insert

## Features

### Voice Input
- **Push-to-talk**: Hold the microphone button while speaking
- **Auto-detect language**: The keyboard automatically detects your language

### Polishing Levels
Tap the intensity indicator to change polishing level:
- **Light**: Filler removal only
- **Standard**: Grammar and logic improvements
- **Deep**: Full rewrite with comprehensive improvements

### Translation
1. Tap the translate button
2. Set your target language in settings
3. Voice input will be automatically translated

### Settings
Access keyboard settings by tapping the gear icon:
- Input language
- Translation language
- Polishing intensity
- Microphone selection

## Troubleshooting

### Keyboard not appearing in list
- **iOS**: Restart the device and try again
- **Android**: Check that the keyboard is enabled in system settings

### Voice input not working
- Ensure microphone permission is granted
- Check that no other app is using the microphone
- Try restarting the app

### Text not inserting
- **iOS**: Ensure Full Access is enabled
- **Android**: Check that Talkute is set as an input method

### Language detection issues
- Set your primary language in keyboard settings
- Speak clearly and avoid mixing languages

## Privacy

The Talkute keyboard:
- Only sends audio when the microphone button is pressed
- Does not log keystrokes
- Processes voice on secure servers
- Does not sell your data
